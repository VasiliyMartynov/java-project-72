Описание

Page Analyzer – сайт, который анализирует указанные страницы на SEO пригодность.

## Requirements

* JDK 20
* Gradle 8.2
* GNU Make

## Setup

```bash
make setup
```

## Run server

```bash
make start
# Open http://localhost:3000
```
